Guía de Uso

¿Qué es un controlador PID?

Un controlador PID (Proporcional-Integral-Derivativo) es un mecanismo de control que calcula la diferencia entre un valor deseado y uno medido, y aplica una corrección basada en tres parámetros:

  • Kp (Proporcional): Reacciona al error actual
  • Ki (Integral): Corrige errores acumulados
  • Kd (Derivativo): Anticipa errores futuros

Cómo usar este simulador

  1. Configura la planta (sistema a controlar) con la función de transferencia
  2. Ajusta los parámetros PID usando los deslizadores
  3. Selecciona la señal de referencia deseada
  4. Observa la respuesta del sistema y las métricas de desempeño
  5. Usa la herramienta de sintonía para encontrar parámetros óptimos
